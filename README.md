# Flask Backend Learning Project

This is a backend learning project built with Flask

## Features

- Basic Flask server
- JSON responses
- Dynamic routes
- Virtual environment setup
- Git version control

## How to run
1. Clone the repo
2. create virtual environment:
    python -m venv venv
3. Activate it:
    venv\Scripts\activate
4. Install dependnecies:
    pip install -r requirements.txt
5. Run the app:
    python app.py

Server runs at:
http://127.0.0.1:5000/
